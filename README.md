## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/uptime.git
cd uptime
go build -o uptime .
./uptime --help
```

## 🚀 Quick Start

```bash
./uptime -h
```

## 🧪 Examples

```bash
# Example 1
./uptime status

# Example 2  
./uptime --verbose

# Example 3
./uptime help
```
