## Uptime

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/uptime.git
cd uptime
go build -o uptime .
```

### Usage

```bash
./uptime --help
```

---

## Quick Start

```bash
./uptime [command]
```
