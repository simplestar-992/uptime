# UPTIME

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/uptime.git
cd uptime
go build -o uptime .
```

## Usage

```bash
./uptime --help
```
